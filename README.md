# Objective:
This assignment will help you develop a foundational understanding of N-grams and their implementation using the Natural Language Toolkit (NLTK). You will create N-grams for various values of N using a sample text dataset.
---

# Scenario:
Imagine you are a junior Data Scientist at Linguistics Inc., a company specializing in natural language processing (NLP) technologies. Your task is to explore the concept of N-grams, which are contiguous sequences of N items from a given text sample. This foundational task will help you understand how text can be represented for various NLP applications such as text generation, predictive typing, and language modeling.
---

# Task:
Create N-grams for various N across multiple sentences using at least 1 or 2 sentences to apply multiple N values.

- Generate N-grams: Create N-grams for various values of N (e.g., 1, 2, 3). Use NLTK’s built-in functions or manually implement the logic to generate these N-grams.

- Python Code: Submit a Python script or Jupyter Notebook with a well-commented code implementing the following:
